CLASS INFORMATION

Description

Introduction to application protocols for large scale distributed systems including object request brokers, asynchronous messaging, and Web services. Lab is based on using protocols to build distributed systems. Prerequisites: Java programming or instructor consent. Computer Engineering and Software Engineering majors only. Corequisites: CMPE 272
